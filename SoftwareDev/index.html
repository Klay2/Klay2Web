WalkingDistance


-WalkingDistance is an Android app that sends users promotional materials depending on their GPS location. I am working on a production level release of this app with my partner, Grisha Kamishnakov. LINK AND SPELLING!!


Details:


-WalkingDistance Details: The App started out as a project for my mobile programming course at the UW-Madison. The primary use of the app is intended to get potential clients walking by businesses on the street into stores. On the original project I created 70% of the Object Model code comprised of the model object design and list adapter code design. I created a background service that sets up and receives GPS updates via a dedicated handler thread.


I also created the code in the Main Activity that upon receiving coordinates from the background thread asynchronously sends an HTTP message to a Java servelet that runs a database query using these coordinates to get corresponding geo-linked promotion objects. I used Heroku to create a starter Java Servelet and modified it to run the query and return JSON converted Objects. For the production scale app so far I have created a more robust background service that downloads promotions and uses a cache file so that it can download promotional Objects while the Main Activity of the App is paused.
